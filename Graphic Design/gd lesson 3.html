You are just changing the background for each image, so you can reuse the same event info.  Please make sure you submit your images in a jpeg file (Save for Web > JPEG) 1) The first image should show the brush being used as a background. 2) The second image should show the brush being used as a background and incorporating the gradient map tool. 3) The third image should show the brush colorized by hand.
