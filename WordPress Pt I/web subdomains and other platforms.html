We will go over how “sub-domains” and what functions they can serve. You will learn how to create them in cPanel and how to install software onto them. Possible applications of sub-domains are creating discussion forums, Moodle for managing a class, or individual blogs for multiple users.

## Outcomes for this lesson:
  • Understand how to quickly install a new platform on your domain
  • Explore best practices for web design
