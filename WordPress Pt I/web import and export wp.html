Sometimes you need to move a lot of content from one WordPress site to another. Conventional wisdom tells you to copy paste all you posts and pages, but that would take forever. We want something quicker. That's why we have an import and export function built into WordPress. ## Exporting Content This part is really easy. Just go to your menu and choose "Tools>Export". ![image](http://labs.everettprogram.org/wp-content/uploads/2013/04/Export.png) You can choose to export all content, or just pages or posts. Click "Download Export File" and you'll get an XML file. ## Importing Content This has more steps, but is also extremely easy. If you would like to follow along, downloadÂ WP Lab Easy Content which is contains some test posts and pages as well as a custom menu containing links to the CollegeBoard.org pages for every California public college. If you just want that custom menu, download CA Public Colleges Menu file. **Note:** These files are .zip files. You must open them to get to the XML file you need. ### Step 1: Download the WP Importer Go to "Tools>Import" and select WordPress from the list of importers. ![image](http://labs.everettprogram.org/wp-content/uploads/2013/04/wp-import-1.png) This will prompt a download confirmation screen to pop up. Click Install Now. Don't worry about security, this is made by WordPress.

![image](http://labs.everettprogram.org/wp-content/uploads/2013/04/wp-import-2.png)

### Step 2: Select Your File To Import Simply choose the file you want to import. ![image](http://labs.everettprogram.org/wp-content/uploads/2013/04/wp-import-3.png) ### Step 3: Determine What User to Publish As All WP posts and pages need to be attributed to an author. You have the option of creating a new user for these posts to be attributed to, or you can just assign an existing user to take credit for them. ![image](http://labs.everettprogram.org/wp-content/uploads/2013/04/import-4.png) ### Step 4: You're Done! You may see a bunch of notices that "Failed to import.....". That just means that it couldn't download attached image files. You may have to move over images manually.
