To get them for Skitch onto your desktop so that you can send them to us, you just need to drag them there. Either from the annotation window or the screenshot manager. ## Mac Apple computers have an easy, built-in screenshot tool that are already bound to your keyboard and will work in any application. Here are the controls:
  • ⌘ + Shift + 3: Takes a screenshot of the screen, and saves it as a file on the desktop
  • ⌘ + Shift + 4: Then select an area: Takes a screenshot of an area and saves it as a file on the desktop
To change the keyboard shortcuts, go to "**System Preferences > Keyboard > Shortcuts**"  and then select "Screen Shots" from the left sidebar. By default, screenshots save directly to your desktop. If you want the screenshot images to save in a location other than the desktop, check out this guide. There are lots of neat tricks you can do with the native screen capture in OS X. ### To take a screen shot and save it as a picture
  1. Click the window you want to capture. Press Alt+Print Screen by holding down the Alt key and then pressing the Print Screen key. The Print Screen key is near the upper-right corner of your keyboard. (Depending on the type of keyboard you have, the exact key names on your keyboard may vary slightly.)
    ### Note
    • You can take a screen shot of your entire desktop rather than just a single window by pressing the Print Screen key while holding down the Alt key.
  2. Click Start, click Accessories, and then click Paint.
  3. In the Paint window, click Edit, and then click Paste.
  4. When the image appears in the Paint window, click File, and then click Save As.
  5. In the Save As dialog box, in the File name box, type a name for the screen shot, and then click Save.
You can now print or email the saved screen shot just like you would any other picture. To take a shot of a specific area of the screen press 'Ctrl' + 'Shift' + 'Windows Switcher' # Compressing (Zipping) Files You will often be asked to "compress" or "archive" or "**zip**" a folder before submitting it. These terms all refer to the same action, which is to create an archive file. Basically, it's a means of combining several files and folders into a single file that's easier to send to someone else. Sometimes a homework assignment will require you to submit multiple files. By putting them all into a folder and **zipping** it up, you can submit a single file rather than uploading them all one by one. ## Mac Put all the desired files into a folder and give it a name. Right-click (control-click) on the folder and choose  Compress "[file name]" ## Windows Right-click on the folder you want to zip and select Send To > Compressed (zipped) Folder. # Submitting Work in the Correct Format One of the biggest issues students have is with submitting their work in unreadable formats. This is obviously a problem because we can't read what's you're giving us. Here are the most common conversions you may need to do. ## Converting Pages to Word Format If you have a Mac and use Pages.app as your word processing app, please understand that no one can open the files you send them. Files created in Pages.app are saved with the .pages extension. These files can *only* be opened if you have Pages.app, which most people do not. **If you submit work in .pages format, it will count as not having been submitted and will be marked off (this applies to the Tuesday class as well). **If you must use Pages.app, then you need to know how to save in a format that's actually useful. Luckily it's easy to do. Go to the **File > Save As **and then check the box to save a copy as Word Document. Make sure you select the .docx format since it's the newest and most portable version. ## When You Use Google Docs Google Docs is a great zero-cost, simple, cloud based word processor, but we find a lot of people don't know how to share their work. ### Don't Upload a Google Doc from your Drive. The files in your Google Drive are actually document files. They're just references to a location in your Drive. They can't be opened by anyone but you. ### Don't Send us the URL for your Doc. For one, nobody who does this ever remembers to allow others to view the doc so we can't see it. Secondly, when you submit an assignment, it is what you did up to the deadline. By sending us a URL to your live, working document, you're not submitting a complete, polished assignment. ### Do Download it as a Word Document or PDF. It's easy. In the Doc, just go to "**File > Download as**"
